# Radio Blueprint - 无线电台站设计蓝图

一个基于 React 和 React Flow 构建的专业级无线电仿真系统可视化工具。它可以帮助工程师和火腿(HAM)们快速构思和设计无线电链路。

## ✨ 特性

- **多样化组件库**：
  - 收发设备：电台、收发分离变频器、信号发生器、GPSDO。
  - 无源器件：定向耦合器、双向耦合器、功分器、负载。
  - 附件：各型号天线、测试仪器。
- **智能化连线系统**：
  - **接口校验**：自动识别连接头类型（同轴 COAX、DC 电源、PTT、信号），防止错误连接。
  - **正交布线**：所有连线自动以直角形式弯折，整洁美观。
  - **可拖拽转点**：选中连线即可拖动转角位置，灵活规避干涉。
- **专业级交互体验**：
  - **组件旋转**：支持 360 度旋转，方便各种拓扑布局。
  - **动态端口**：功分器可根据配置动态改变输出端口数量。
  - **主题切换**：支持深色/浅色配色方案，保护视力。
  - **自动保存**：实时将设计进度保存至浏览器本地存储，防止数据丢失。
  - **导出导入**：支持设计方案导出为标准 JSON 文件或 PNG 高清图片。

## 🚀 快速开始

### 预求环境
- Node.js (推荐 v18+)
- Bun (推荐) 或 NPM

### 安装与运行
1. 克隆/进入项目目录
2. 安装依赖：
   ```bash
   bun install
   ```
3. 启动开发服务器：
   ```bash
   bun run dev
   ```
4. 访问 `http://localhost:5173` 开始设计。

## 🛠️ 技术栈
- **核心框架**: React 18
- **画布引擎**: [@xyflow/react](https://reactflow.dev/) (React Flow)
- **状态管理**: React Hooks
- **样式方案**: Vanilla CSS + Inline Styles
- **图标/设计**: Emoji + CSS Gradients
